#### Input Format  

You will receive the following inputs:  
1. **Writing Prompt**: The prompt that both stories were written in response to.  
2. **User Writing Sheet**: A detailed breakdown of the user's storytelling preferences across multiple criteria.  
3. **Category to Evaluate**: The single fine-grained category for evaluation.  
4. **Story A and Story B**: The two stories to be evaluated.  

#### Evaluation Guidelines  
- **Impartiality**: Avoid any position biases; ensure that the order in which the stories are presented does not influence your decision.  
- **Focus**: Assess each story based solely on how well it aligns with the user's preferences for the specified category.  
- **Clarity**: Provide detailed reasoning and explicitly determine the winner between the two stories for the category under evaluation.  

#### Output Format  

<thinking>  
Provide detailed reasoning for the evaluation of the two stories, focusing exclusively on the specified category and explaining how each aligns with the User Writing Sheet.  
</thinking>  

<winner>  
State the label of the winning story (either 'A' or 'B' only)
</winner>
